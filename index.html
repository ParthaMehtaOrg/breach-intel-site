How It Works

Three commands. Full coverage.

No SDK wrapping, no agent modifications, no middleware injection. Install once, monitor everything.

Install

One pip install drops the policy agent, classifier, and persistent hook onto your system. pip install breach-intel

Hook

The install-hook command writes a sitecustomize.py that auto-patches every Python agent at import time. OpenAI, Anthropic, LangChain — all intercepted.

Run

Start your agents normally. Every LLM response is scanned in <1ms by the rule-based classifier. Breaches log to DB, JSONL, and webhooks — immutably.

Capabilities

Everything you need for agent compliance

🔍

Rule-Based Classification

Deterministic breach detection in <1ms. No LLM in the critical path. Pattern matching for PII, PHI, financial data, and 40+ violation types.

🧠

Agentic Deep Analysis

Optional async LangGraph pipeline powered by Claude. Six-node analysis graph adds context, rationale, and regulatory mapping — without blocking.

🔗

SHA-256 Hash Chain

Every breach record is checksummed and chained. Integrity scanning runs every 6 hours. If a single record is tampered, the chain breaks.

📊

Live Dashboard

Single-file React SPA served at /dashboard. Breach counts, severity breakdown, agent heatmaps, live feed, and click-to-inspect detail views.

🔌

Zero-Touch Auto-Attach

Persistent sitecustomize.py hook patches all AI frameworks at import time. Your agent code stays untouched. Works with OpenAI, Anthropic, LangChain.

🚨

Webhook Alerts

CRITICAL breaches fire instantly to Slack, Discord, or generic webhooks. Configure thresholds and channels per vertical.

📝

Immutable Dual-Write

Every breach writes to PostgreSQL/SQLite AND real-time JSONL files. Two independent audit trails. Ship JSONL to your SIEM.

🏗️

Auto-Scaling

When event load exceeds threshold (default 500/min), the policy agent spawns additional instances via Docker, K8s, or subprocess.

🔐

Multi-Tenant Auth

Scoped API keys with tenant isolation. Rate limiting at 300 events/min per key. Admin keys exempt. Full RBAC.
